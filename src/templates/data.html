Data

This page lists all current GeneWeaver species, gene identifier sources, and microarray platforms. We cannot include certain identifiers such as RefSeq or GenBank due to scalabilty concerns. For example: Entrez Nucleotide (which includes RefSeq and GenBank) has over 400x more identifiers than Entrez Gene for Mouse. If we included these identifiers, uploads and other gene queries would take much longer to complete, resulting in a painful user experience.

New Species

If you would like to request a new species, please email geneweaver@gmail.com, and include the NCBI Taxonomy ID (found here). Please make sure that the Taxid you select has directly linked Entrez Gene records. This information is found in the the table below. For example, Saccharomyces cerevisiae (Taxid 4932) has no direct Gene links, but S. cerevisiae S288c (Taxid 559292) has over 6,000.
